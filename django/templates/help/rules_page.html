Normas generales

  1. Regla no escrita.
    2. {% comment %}
  2. Un moderador no puede inclumpir las normas o, por inacción, permitir que las normas sean incuplidas.
  3. Un usuario debe obeceder las órdenes dadas por un moderador, excepto si estas órdenes entrasen en conflicto con las normas generales.
  4. Los moderadores deben procurar que haya un buen ambiente, en la medida que hacerlo no entre en conflicto con las normas generales.
    5. {% endcomment %}
  5. No se permite:
    • Insultar, despreciar, hacer mofa o publicar datos privados de cualquier persona concreta.
    • Enlaces a contenido ilegal.
    • Hacer spam o interferir con el funcionamiento normal del sitio.
    • Explotar bugs o fallos de seguridad sin avisar con antelación a un administrador.
  6. Si a alguien se le ocurren más maneras de hacer el capullo puede que cambien las normas.
